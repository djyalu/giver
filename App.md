## Vercel 배포 절차
1. GitHub 저장소에 코드를 푸시합니다.
2. Vercel에서 새 프로젝트를 생성하고 저장소를 연결합니다.
3. **Environment Variables** 설정 섹션에서 다음을 추가합니다:
   - Key: `API_KEY`
   - Value: [Google AI Studio에서 발급받은 키]
4. 배포 버튼을 클릭합니다.
